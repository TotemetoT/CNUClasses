# Overview

This in-class participation exercise calculations using Numpy, and plotting
using MatPlotLib.

It is also intended to illustrate the power and underlying beauty of math!

We will also be discussing the concept of "numerical methods", specifically
"numerical integration" using Euler's method.

Students must fork to their `firstname.lastname.yy-cpsc250-semesterYear` group, and clone the forked project using Gitlab and Git from the command line.  Complete the exercises using PyCharm by following along with in-class demonstrations.  

Submit all the plots to Scholar in one group for a participation based grade.
Minimal typing is required today, but you will need to edit the code and run the scripts to
generate all of the plots.


## Homework

You should be able to complete this homework in under 45 minutes.

Set a timer and practice under timed conditions.
You might want to practice a couple of times.

* homework_plot.py

  Shell code is given in `src/homework_plot.py`.

  This part requires you to write some code to do some numpy calculations,
  and then plot the data as described in the doc-string.

  Use numpy *NOT* lists and loops!

  Submit your plot to scholar with the other W10 plots


* homework_inherit.py (20 points)
  * Write a new class using inheritance
    * directions given in homework_inherit.py
    * create the new class in a new file of your creation
  * Use the class you wrote as directed in `homework_inherit.py` main
    * create instances in list
      * Note: Make up your own data to create instances, you are NOT allowed
        to use data from file for this part.
  * Write function that uses list of classes as described in doc-string

* homework_file.py
  * File I/O and Exception handling
  * read data from file and create list of classes instances
    * Start with `short_data.csv`,
    * you must the `full_data.csv` for full credit
      * ignore bad data, but continue to process remaining lines
  * the instance types must be the designated class
  * fix code in main to calculate the total bread cost given list
     * Hint: A function from `homework_inherit` might be useful
