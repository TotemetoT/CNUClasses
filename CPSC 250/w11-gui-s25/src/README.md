# Overview

Students must fork to their `firstname.lastname.yy-cpsc250-semesterYear` group, and clone the forked project using Gitlab and Git from the command line.  
Complete the exercises using PyCharm.

The first in-class participation exercise will give us some practice with
Graphical User Interfaces (GUIs) using tkInter.

https://docs.python.org/3.7/library/tkinter.html

The second part will cover pygame development if time permits.

https://www.pygame.org


****
Project Coding Exercises
====

## In class
### GUI Development
* Demonstrate event driven model in `gui_intro.py`
* Update text field in `gui_grid.py`
* Write a helper gui tool in `helper_gui.py`
    * This should take a python function/object that has a doc_string and print the docstring to the text field.
    * Follow instructions in the doc string and in class
### PyGame
* Work on `demo_game.py` to demonstrate the basics of pygame
