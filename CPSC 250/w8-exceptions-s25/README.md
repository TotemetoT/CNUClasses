Project Coding Exercises
====


### Note: There are several "Demo" files for you to review in the given folder.

Use those for your edification.

## In Class - Exceptions
* Define a new Exception called ``MyVeryOwnError`` in ``my_very_own_error.py``
* Write the static `throw_something` method according to the directions in doc string of `exceptions.py`
* Write the static `catch_something` method according to the directions in its doc string of `exceptions.py`

* In `advanced_list.py` we will write a class AdvancedList that will inherit from the built-in list class.
    * AdvancedList will demonstrate operator overloading to handle multiple types.
    * AdvancedList will also demonstrate exception handling.
    * In addition it will be a prelude to using NumPy arrays that will be used in the following week.
