# Overview

This in-class participation exercise with recursion.

Students must fork to their `firstname.lastname.yy-cpsc250-semesterYear` group, and clone the forked project using Gitlab and Git from the command line.  
Complete the exercises using PyCharm by following along with in-class demonstrations.  


****
Project Coding Exercises
====

## In Class - Recursion
* In `factorial.py` we will write methods to calculate factorial using a loop (150 style) and recursion (250 style).
* In `fibonacci.py` we will write methods to calculate the value of a Fibonacci Sequence at a given index.
    * https://www.mathsisfun.com/numbers/fibonacci-sequence.html
* We will work through several examples of recursion in `recursion_practice.py`
* `fibonacci_demo.py` is written to provide further insight into the operation of recursion, and the "call stack" used during processing.
* In `towers_of_hanoi.py` we will use recursion to solve the famous towers of Hanoi problem.
* Write a function to print out all exceptions in `exceptions_inheritance.py`

## Homework - Recursion
