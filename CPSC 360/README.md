CPSC 360 - Programming Language Concepts

Throughout this course several different programming languages were looked at, some more obscure than others. Some of the languages that were discussed include Rust, Haskell, Fortran, C#, Java, etc. While the languages were a main focus the different syntax, data types, and functions (recursion, polymorphism, etc.) were discussed as well. There were 5 main languages that were discusses in this course, those being:

- C#
- C++
- Haskell
- Prolog
- PHP ( + HTML, CSS, and JavaScript)
