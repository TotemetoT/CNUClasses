CPSC 250L - Programming for Data Manipulation Lab

This course provides an application the the concepts that were taught in CPSC 250, Programming for Data Manipulation. Throughout this course I was able to further my understanding of the topics discussed in CPSC 250 by seeing how the concepts work in each lab. Some of the topics discussed include:
- Working with Lists & Dictionaries
- String Formatting
- File I/O
- Classes & Polymorphism
- Instances
- MatPlotLib/Plots
- Numpy
- Python GUI
- PyGame
